1
Yao et al. (2022) — "ReAct: Synergising Reasoning and Acting in Language Models" FREE
The foundational paper for the Thought→Action→Observation loop that powers most AI agents. Princeton + Google Brain.
🔗 arxiv.org/abs/2210.03629
2
Anthropic — Tool Use (Function Calling) Documentation FREE
Official guide for Claude's tool use API — defining tools, handling tool_use blocks, multi-turn tool loops, and best practices.
🔗 docs.anthropic.com/en/docs/build-with-claude/tool-use
3
LangChain — Agents Documentation FREE
Comprehensive guide to LangChain agents, tools, AgentExecutor, and the agent loop implementation in Python.
🔗 python.langchain.com/docs/concepts/agents
4
Anthropic — "Building Effective Agents" Research Post FREE
Anthropic's definitive guide on when to use agents, agentic patterns (augmented LLM, routing, parallelisation, orchestrators), and safety considerations.
🔗 anthropic.com/research/building-effective-agents
5
Wang et al. (2023) — "A Survey on Large Language Model based Autonomous Agents" FREE
Comprehensive survey of LLM agent architectures: profiling, memory, planning, and action modules. 80+ papers reviewed.
🔗 arxiv.org/abs/2308.11432
6
CrewAI Documentation FREE / Open Source
Framework for building multi-agent systems with role-based agents, tasks, and crew orchestration. Python-first, production-ready.
🔗 docs.crewai.com
7
DeepLearning.AI — "AI Agents in LangGraph" Short Course FREE
Free course by Andrew Ng covering agentic patterns, memory, multi-agent systems, and LangGraph implementation.
🔗 deeplearning.ai/short-courses/ai-agents-in-langgraph
8
Microsoft AutoGen Documentation FREE / Open Source
Microsoft's multi-agent conversation framework enabling complex workflows with conversational agent patterns.
🔗 microsoft.github.io/autogen
9
Lilian Weng — "LLM Powered Autonomous Agents" Blog Post FREE
OpenAI researcher's definitive deep-dive on agent components: planning, memory, tool use, and case studies. Highly recommended.
🔗 lilianweng.github.io — LLM Powered Autonomous Agents
